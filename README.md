# Steam Game Analytics & Engagement Analysis

## Overview

This project analyzes the top Steam games dataset to understand player engagement and popularity trends.

## What I Did

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Data visualization using Python
* Created retention ratio for engagement analysis

## Tech Stack

* Python
* Pandas
* Matplotlib
* Seaborn
